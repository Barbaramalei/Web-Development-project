Instructions on how to run the system
1. Make sure you have XAMPP server installed in your machine.
2. Copy paste the project folder into the 'htdocs' XAMPP folder.
3. Start the XAMPP control panel, then start Apache and MySQL.
4. Still on the XAMPP control panel, click MySQL 'admin' tab which opens in a browser.
5. Create a database with the name 'inventory'
6. Open a new tab on your browser and type 'localhost/inventory'

Note: The default admin user has the following login credentials:
Username: admin
Password: Admin123
